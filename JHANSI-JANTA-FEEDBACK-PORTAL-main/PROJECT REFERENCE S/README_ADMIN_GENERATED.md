## 🚀 Quick Start (5 Minutes)

### Step 1: Create Admin Account
```bash
mongosh
use janata_portal
db.users.updateOne({email: "your@email.com"}, {$set: {role: "admin"}})
```

### Step 2: Log In
- Go to http://localhost:3000/login
- Enter credentials

### Step 3: Access Admin Panel
- Navigate to http://localhost:3000/admin
- **Done!** 🎉

## 📊 What Admins Can Do

### Dashboard Tab 📊
- View 6 statistics cards (total grievances, users, resolved, pending, rejected, avg time)
- See status distribution chart
- Check recent activity feed
- Monitor system health

### Grievances Tab 📋
- Search grievances by title, description, or citizen ID
- Filter by status (Pending, In Progress, Resolved, Rejected)
- Filter by category (General, Infrastructure, Health, Education, Safety)
- View full grievance details
- Update grievance status
- Add responses and comments
- Delete grievances if needed

### Users Tab 👥
- View all registered users
- Search users by name or email
- See user details (name, email, phone, role, join date)
- Delete user accounts
- Color-coded role display
